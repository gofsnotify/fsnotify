# Project Conventions

## Reference Implementation
- This is a separate project from fsnotify/fsnotify. Do not consult its source.

## Commits
- Write messages in English.
- Keep them sufficient but not too long.
- Append `Co-Authored-By: Claude <noreply@anthropic.com>` as a trailer.

## Issues
- Write in English.

## Pull Requests
- Description only, in English. Do not use section headings like `# Summary`.
- For performance-related PRs, include a before/after comparison table.
